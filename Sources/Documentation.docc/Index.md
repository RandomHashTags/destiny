# Destiny Documentation

Destiny is a standalone, high-performance and lightweight http networking framework. The main advantages of this library over Hummingbird and Vapor comes from the native usage of Concurrency, Swift Macros, industry leading runtime performance, tiny binary size, supporting Embedded, and no legacy code.
