# Destiny Server

Destiny uses a Swift Macro to generate an optimal Server and has a `NonCopyableHTTPServer`.

## Macros

See [Macros.swift](https://github.com/RandomHashTags/destiny/blob/main/Sources/DestinySwiftSyntax/Macros.swift)

## HTTPS

Destiny does not support https/tls/ssl yet.

## Web Sockets

Destiny does not support web sockets yet.
